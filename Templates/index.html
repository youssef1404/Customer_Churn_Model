Classification Model for Customer Churn Detection

Problem Definition
  • The dataset consits of 10 features
  • The target is Churn (Customers who left within the last month)
  • Features are ('CreditScore', 'Geography', 'Gender', 'Age', 'Tenure', 'Balance', 'NumOfProducts', 'HasCrCard', 'IsActiveMember', 'EstimatedSalary')
  • The final Chosen model is XGBoost and the Evaluation metric is F1-score
  • XGBoost achieved about (F1-score=71.825%) on train dataset and (F1-score=62.055%) on test dataset.
